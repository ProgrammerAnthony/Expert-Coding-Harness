# React最佳实践技能

基于React官方推荐和行业通用最佳实践打造的React开发指导工具，覆盖组件设计、Hooks使用、状态管理、性能优化、Next.js开发全场景。

## 核心能力
- 组件设计指导：合理拆分组件、设计Props、提升复用性
- Hooks最佳实践：避免常见Hooks使用错误，发挥Hooks最大价值
- 状态管理选型：根据项目规模选择最合适的状态管理方案
- 性能优化：针对性解决React重渲染、列表卡顿等性能问题
- Next.js专项指导：App Router、渲染模式选择、数据获取优化
- 规范制定：团队React开发规范、代码规范制定

## 适用场景
- React项目从零开始架构设计
- React代码重构、优化
- React开发过程中的问题解答
- 团队React开发规范制定
