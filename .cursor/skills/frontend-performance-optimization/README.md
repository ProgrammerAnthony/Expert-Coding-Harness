# 前端性能优化技能

系统性的前端性能优化解决方案，覆盖加载性能、运行时性能、核心Web Vital指标全场景优化。

## 核心能力
- 精准性能瓶颈定位：基于性能数据分析找到核心问题
- 分级优化方案：按收益成本排序，优先实施高性价比优化
- 全场景优化支持：首屏加载、交互流畅度、打包体积、内存优化全覆盖
- 可量化的优化效果：每个优化方案都标注预期收益和验证方法

## 适用场景
- 页面加载慢、白屏时间长
- 交互卡顿、响应不及时
- LCP/FID/CLS等Web Vital指标不达标
- 打包体积过大，加载速度慢
- 移动端性能体验差
- 项目上线前性能验收

## 使用方式
提供性能分析报告（Lighthouse、Chrome Performance、打包分析报告等），即可得到结构化的优化方案。
